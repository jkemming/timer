Installation

timer.mob.sh is a shared team timer for the tool mob.sh.

Define your team room, by using a non-guessable string. Configure mob.sh by setting an environment variable (e. g. in your ~/.zshrc file).

export MOB_TIMER_ROOM=myteam-327266

The mob tool then sends a request to the timer.mob.sh server, including the current interval and your configured Git username.

You can overwrite the username by setting this environment variable.

export MOB_TIMER_USER=override-git-user

Use mob as normal:

  • mob start 10
  • mob timer 10
  • mob break 5

All team members can view the timer at https://timer.mob.sh/myteam-327266
